In general colors are represented in 4 ways . They are as folows:

color name
the word says by name
HEXA CODE
  • represented as #rrggbb(first2-red,next-green,lastly-blue)
  • minimum value is 0(zero) and max.value is' f '(15)
  • red=>#ff0000
    • RGB VALUES
  • represented as rgb(R,G,B)
  • minimum value is 0(zero) and max.value is'255 '(255)
  • red=>rgb(255,0,0)
  • rgba(R,G,B,a[opacity])
  • red with light intensity=>rgba(255,0,0,0.5)
    • HSL VALUES
  • represented as hsl(Hue,Saturation percentage,Lightness percentage)
  • minimum value is 0(zero) and max.value is' 360'(360)
  • percentage is between 0-100%
  • light-green=>hsl(121,50%,60%)
